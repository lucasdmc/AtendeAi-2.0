## ⚙️ Como Configurar

1. **Copie seus agentes** para a pasta `.cursor/agents/`
2. **Ajuste as configurações** em `.cursor/settings.json`
3. **Personalize os templates** em `.cursor/templates/`
4. **Reinicie o Cursor** para carregar as configurações
